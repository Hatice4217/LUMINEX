## Proje Genel Bakış

Luminex, sağlık profesyonellerinin ve yöneticilerin ihtiyaçlarını karşılamak üzere tasarlanmış kapsamlı bir web tabanlı platformdur. Aşağıdaki işlevleri sağlar:
- Kullanıcı yönetimi (Yöneticiler, Doktorlar, Hastalar)
- Randevu planlama ve yönetimi
- Doktor uygunluğu ve profilleri
- Hasta sağlık geçmişi ve reçeteleri
- Laboratuvar ve radyoloji raporu yönetimi
- İdari raporlama ve departman yönetimi

## Kurulum ve Yükleme

Luminex projesini yerel olarak kurmak için aşağıdaki adımları izleyin:

1.  **Depoyu klonlayın:**
    ```bash
    git clone [depo-url]
    cd LUMINEX
    ```
2.  **Bağımlılıkları yükleyin (varsa, örn: geliştirme araçları veya derleme süreçleri için):**
    ```bash
    npm install
    ```
    *(Not: Bu proje ağırlıklı olarak istemci tarafı HTML/CSS/JS gibi görünmektedir. `npm install`, lintleyiciler, paketleyiciler vb. geliştirme bağımlılıkları için olabilir. Tamamen statikse, bu adım atlanabilir veya basitleştirilebilir.)*

3.  **Uygulamayı açın:**
    Web tarayıcınızda `login.html` dosyasını açmanız yeterlidir.
